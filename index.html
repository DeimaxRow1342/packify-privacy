PackifyDelivery

Tu solución confiable para el seguimiento de paquetes y envíos

1. Información que Recopilamos

  • Email y nombre de usuario para autenticación mediante Firebase Authentication
  • Datos de ubicación en tiempo real para servicios de mapas y seguimiento
  • Información de paquetes (destinatario, dirección, estado de entrega)
  • Datos de uso de la aplicación y preferencias del usuario
  • Metadatos técnicos (tipo de dispositivo, sistema operativo, versión de la app)

2. Uso de la Información

  • Proveer el servicio de tracking de paquetes en tiempo real
  • Mejorar la experiencia de usuario y funcionalidades de la aplicación
  • Comunicarnos con usuarios sobre el estado de sus envíos
  • Personalizar el servicio de entrega y recomendaciones
  • Garantizar la seguridad de la plataforma y prevenir fraudes

3. Almacenamiento y Seguridad

  • Los datos se almacenan en Firebase (Google Cloud Platform) con encriptación en reposo
  • Todas las comunicaciones utilizan HTTPS con encriptación TLS 1.2+
  • Las contraseñas están protegidas mediante hash SHA-256
  • Cumplimos con normativas de protección de datos internacionales
  • Los datos de ubicación se retienen solo durante la sesión activa, salvo consentimiento explícito

4. Compartición de Datos

  • NO compartimos datos personales identificables con terceros
  • Los datos solo se utilizan para funcionalidades internas de la aplicación
  • Información anónima y agregada para análisis de uso y mejoras
  • Integración con Mapbox para servicios de mapas (datos de ubicación anonimizados)

5. Tus Derechos

  • Acceder y descargar tus datos personales en formato legible
  • Solicitar la eliminación completa de tu cuenta y datos asociados
  • Exportar tus datos a otro servicio mediante formatos estándar
  • Revocar permisos de ubicación y notificaciones en cualquier momento
  • Rectificar información incorrecta o desactualizada

6. Control de Tus Datos

Valoramos tu privacidad y te ofrecemos control total sobre tu información personal.
