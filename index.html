Política de Privacidad - Packify Delivery

1. Información que Recopilamos

• Email y nombre de usuario para autenticación
• Datos de ubicación para servicios de mapas
• Información de paquetes (destinatario, dirección, etc.)
• Datos de uso de la aplicación

2. Uso de la Información

• Proveer el servicio de tracking de paquetes
• Mejorar la experiencia de usuario
• Comunicarnos con usuarios sobre sus envíos
• Personalizar el servicio de entrega

3. Almacenamiento y Seguridad

• Los datos se almacenan en Firebase (Google Cloud Platform)
• Las contraseñas están encriptadas con SHA-256
• Cumplimos con leyes de protección de datos internacionales
• Los datos de ubicación se usan solo durante la sesión activa

4. Compartición de Datos

• NO compartimos datos personales con terceros
• Los datos solo se usan para funcionalidades de la app
• Información anónima para análisis de uso

5. Tus Derechos

• Acceder a tus datos personales
• Solicitar la eliminación de tu cuenta
• Exportar tus datos
• Revocar permisos en cualquier momento
